How to get your bot's token

What is a token

Bot tokens are how we link our program to our bot's user ID

As long as a user has the bot token they can manipulate what the bot does

Make sure that you don't share your token with anyone

Getting your token

To get our token we go to: https://discordapp.com/developers

We have to first create an app by clicking "create an app" You can name the app and also give it a profile picture.

From there we can go to the app's setting and create a bot account. There will be a button that will reveal the bot's token. Make sure you copy the token and save it somewhere.

You will also be able to grab some other information from the bot page. For now just leave it be.

Keep the browser window open and go to the next lesson.
