Basics of creating Discord bots

Information

There are various uses for discords bots. We can play games with users, log messages, moderate the chat, play music in voice channels, and do so much more.

In this tutorial you will be able to create a fully functional discord bot that you can run on your own.

Before this tutorial you should have a basic understanding of the language you want to create the bot with. I will be showing you how to create a bot with discord.py but you can create a bot in other languages.

Python

If you want to follow along with the tutorial there are few things you will have to do first.

First you will want to download and install Python 3.6

When Python is finished installing open up the command prompt and run py -m pip install discord.py

Also run py -m pip install asyncio

After these two libraries are finished installing we can leave them alone for the time being.
