Basics of Python Functions

Functions

What is a function? Well a function is this magical device created by the gods of old rome, in order to make the programmer slaves job just a bit easier... In all seriousness, functions are ways to reduce repetitive code. A function is a block of code you write and save to a variable name. Every time you call that variable name, you run the code.

Lets see it in action 


def functionName():
  print('Heya')
functionName()

This is a simple use of functions. We start out with def which tells the program that the next characters are defining a function. functionName(): is the function name. We add () in order to tell the system that we're done naming the function. (The () are also used in a different way but I'll explain this later).

: Tells the program (just like in our conditionals) that the next lines that are indented go along with this function. Now the next indented lines are ran when ever the function is called; Meaning print('heya') is ran when ever we call the function.

Last but not least is functionName() which runs the function.

Passing Arguments to functions

But there is a way to pass variables, or "arguments" to a function! This way you can pass the password into the function to check it! The code for this is:

So what happens if we want to check a password using a function. We'll you will quickly realize that there is no way to check the password using the function because the function does not have access to the password 


def functionName(password):
    if(password=='x'):
      print('success')
    if('password'!='x'):
      print('fail')
functionName('x')

What we have done different from our last function was added the word "password" to the () in the function. Now when you call the function you have to add an argument into the () within the function name. This is why we do functionName('x') because 'x' is the argument we are passing into the function. We can change 'x' to what ever we want.

Now when you add an argument it gets put into the variable defined in the opening line of our funtion. In our case the argument 'x' is put into the variable "password". Since we define the variable in the function start line we can now use it within the function.

You can pass even more arguments into the function. For example:


def start(time, day):
  print('The day is':+day+" and the time is: "+time)
start('12:32', 'Tuesday')
